# 妙启财 EnlightenRich

一款专业的个人记账软件，采用原生 HTML + CSS + JavaScript 开发。

## 功能特点

### 📅 日历视图
- 完整显示星期信息（周一至周日）
- 自动定位并高亮当前日期
- 每日收支金额摘要显示（收入绿色，支出红色）
- 点击日期查看当天详细记录

### 💰 收支记录
- 右下角浮动"+"添加按钮
- 支持选择"收入"或"支出"类型
- 多种分类可选（餐饮、交通、购物、工资、奖金等）
- 自动记录日期（根据用户点击位置自动填写）
- 支持备注信息
- 支持编辑和删除记录

### 📊 统计分析
- 时间维度切换：本月/本年/本周
- 收支趋势折线图
- 收入分类饼图
- 支出分类饼图
- 收入/支出/结余统计卡片
- 数据导出功能

### 💾 数据管理
- localStorage 本地存储
- 数据备份（导出 JSON 文件）
- 数据恢复（导入 JSON 文件）
- 清空所有数据
- 首页显示总余额

## 技术栈
- **前端**: HTML5 + CSS3 + JavaScript (ES6+)
- **图标**: Font Awesome 6.4
- **图表**: ECharts 5.4
- **存储**: localStorage

## 主题配色
- **主色（黄色）**: #FFC107
- **收入色**: #4CAF50
- **支出色**: #F44336

## 使用方法
1. 直接在浏览器中打开 index.html
2. 点击右下角"+"按钮添加记录
3. 点击日历中的日期查看当天记录
4. 点击底部"统计"查看数据分析
5. 点击底部"设置"进行数据备份/恢复

## 数据说明
- 所有数据保存在浏览器 localStorage 中
- 更换浏览器或清除缓存将丢失数据
- 建议定期使用"设置"中的备份功能导出数据
