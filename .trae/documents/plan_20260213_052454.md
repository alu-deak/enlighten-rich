# 《妙启财EnlightenRich》个人记账软件 - 开发计划

## 一、技术选型

- **技术栈**: 原生 HTML5 + CSS3 + JavaScript (ES6+)
- **图表库**: ECharts (CDN引入)
- **图标**: Font Awesome (CDN引入)
- **存储**: localStorage (本地数据持久化)
- **主题色**: 黄色系 (#FFD700, #FFC107, #FFA000等)

## 二、界面设计

### 配色方案
- 主色(黄色): #FFC107 (琥珀黄)
- 强调色: #FFA000 (深琥珀)
- 收入色: #4CAF50 (绿色)
- 支出色: #F44336 (红色)
- 背景色: #FFF8E1 (浅黄)
- 文字色: #333333
- 卡片背景: #FFFFFF

### 布局结构
- 顶部：应用标题 + 当前月份显示
- 主体：日历视图（7列网格，显示完整星期）
- 底部导航：日历 | 统计 | 设置
- 右下角：浮动"+"添加按钮

## 三、功能模块

### 1. 日历视图
- 显示完整月份日历（周一至周日）
- 高亮当前日期
- 每日收支摘要显示（收入绿色，支出红色）
- 点击日期展开当天详细记录

### 2. 收支记录
- 点击"+"按钮弹出选择：收入/支出
- 记录表单：金额、分类、日期、备注
- 分类预设：
  - 收入：工资、奖金、投资、兼职、 others
  - 支出：餐饮、交通、购物、娱乐、教育、医疗、 others

### 3. 统计分析
- 时间切换：年 / 月 / 周
- 收支趋势图（折线图）
- 分类占比（饼图）
- 收支对比（柱状图）

### 4. 数据管理
- localStorage本地存储
- 数据备份（JSON文件下载）
- 数据恢复（文件导入）
- 清空数据功能
